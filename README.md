## Local setup
**Clone repo**
```
git clone https://github.com/Ngoakor12/ngoako.dev.git
```
### Setup frontend
**Install npm packages**
```
npm install
```
**Run frontend**
```
npm start
```
### Setup Sanity
**Install npm package**(in studio)
```
npm install
```
**Run Sanity**
```
sanity start
```
